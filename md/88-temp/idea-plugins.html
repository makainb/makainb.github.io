    IDEA的插件

    老年人凯哥 2021/12/4

    # Rainbow Brackets(彩虹括号)

    你的括号太多，说明代码要优化了，已经不需要这个插件了。

    # CodeGlance3 （代码鸟瞰图）

    没有不要，有了更好

    # GsonFormatPlus (JSON转换)

    json demo转实体类，就是有点智障，还需要调整

    # Translation （翻译）

    从汉字改为土味英语。

    # BackgroundImagePlus （背景图）

    听说用这个插件的有两种人，一种是装B，另一种是男人。

    # Key Promoter X （快捷键提示）

    老年人用了还是忘。新人用了一天，代码没来及写，一直在背快捷键了。

    # SonarLint （代码质量检测）

    自己给自己找不自在。

    # Statistic （代码统计插件）

    每天看一下，欣赏最水的自己。

    # Codota （代码智能提示插件）

    代码智障提示

    # jclasslib （字节码查看）

    这个插件的汉化，是给我整懵逼了。抄家伙，去除汉化
    1 定位jar包.plugs.jclasslib.lib.jclasslib-browser.jar
    2.定位Properties文件 org.gjt.jclasslib.browser.messages
    3.删除文件Browser_zh_CN.properties
